GITHUB PAGES DEPLOYMENT INSTRUCTIONS
===================================

1. DOWNLOAD YOUR PORTFOLIO FILES
   - Download the entire 'build' folder contents
   - This includes: index.html, assets folder, README.md

2. CREATE GITHUB REPOSITORY
   - Go to github.com and create a new repository
   - Name it something like: your-portfolio, portfolio-website, etc.
   - Make it public for GitHub Pages to work

3. UPLOAD FILES
   - Upload ALL files from the build folder to your repository root
   - Your repository should contain:
     * index.html
     * assets/style.css
     * assets/script.js
     * README.md
     * CUSTOMIZATION_GUIDE.md

4. ENABLE GITHUB PAGES
   - Go to repository Settings tab
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Select "main" branch and "/ (root)" folder
   - Click Save

5. ACCESS YOUR WEBSITE
   - Your site will be available at:
     https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME
   - It may take a few minutes to become active

6. CUSTOMIZE WITH YOUR CONTENT
   - Follow the README.md instructions to add your images
   - Replace placeholder content with your portfolio work
   - Update contact information and branding

Your DVD portfolio website is now live and ready to showcase your creative work!
